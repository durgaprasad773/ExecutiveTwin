# Executive Twin - Authentication Version

Complete frontend with login flow and exact Executive Twin design.

## Features
- 🔐 Microsoft-style login screen
- 👤 User profile display with avatar
- 🎨 Exact same design as Executive Twin
- 📧 Real Outlook integration
- 🤖 AI-powered briefing

## Setup

```powershell
cd frontend-auth
npm install
npm run dev
```

Open http://localhost:3000

## Flow
1. **Login Screen** - Click "Sign in with Microsoft"
2. **Dashboard** - See user profile + "Initialize Monday Briefing" button
3. **Briefing** - AI analyzes Outlook data and shows prioritized tasks

## Design
Same exact styling as executive-twin folder:
- Dark mode executive dashboard
- Gradient headers
- Smooth animations
- Professional layout
